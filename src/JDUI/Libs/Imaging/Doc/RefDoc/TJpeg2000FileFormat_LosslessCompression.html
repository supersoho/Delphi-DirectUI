TJpeg2000FileFormat.LosslessCompression Property
TJpeg2000FileFormat Class

Specifies JPEG 2000 image compression type.

Class

TJpeg2000FileFormat Class

Pascal
published property LosslessCompression: LongBool read FLosslessCompression write FLosslessCompression;
Description

If True, saved JPEG 2000 files will be losslessly compressed. Otherwise lossy compression is used. Default value is False. Accessible trough ImagingJpeg2000LosslessCompression option.

ElderImagery.pas
Classes | Constants | Records | Types

This is basic unit of Elder Imagery extension for Vampyre Imaging Library.

Description

It adds support for loading and saving of images and textures from older Bethesda games (like TES2: Daggerfall, Redguard, Terminator: FS, TES: Arena, ...). This unit registers all file formats declared in additional ElderImagery units so its the only unit you need to add to uses clause of your project for Imaging to be able to load/save these new formats using standard loading/saving functions.

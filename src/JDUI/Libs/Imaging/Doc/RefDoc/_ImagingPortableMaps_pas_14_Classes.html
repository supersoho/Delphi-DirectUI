ImagingPortableMaps.pas Classes
Classes
TPAMFileFormat 
Portable Arbitrary Map is format that can store image data formats of PBM, PGM, and PPM formats with optional alpha channel. 
TPBMFileFormat 
Portable Bit Map is used to store monochrome 1bit images. 
TPFMFileFormat 
Portable Float Map is unofficial extension of PNM format family which can store images with floating point pixels. 
TPGMFileFormat 
Portable Gray Map is used to store grayscale 8bit or 16bit images. 
TPortableMapFileFormat 
Base class for Portable Map file formats (or Portable AnyMaps or PNM). 
TPPMFileFormat 
Portable Pixel Map is used to store RGB images with 8bit or 16bit channels.

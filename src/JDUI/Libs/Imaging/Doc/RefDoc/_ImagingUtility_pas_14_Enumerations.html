TFileListOption 
Options for BuildFileList function: flFullNames - file names in result will have full path names (ExtractFileDir(Path) + FileName) flRelNames - file names in result will have names relative to ExtractFileDir(Path) dir flRecursive - adds files in subdirectories found in Path.

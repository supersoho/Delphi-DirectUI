ImagingExtras.pas
Constants

This is helper unit that registers all image file formats in Extras package to Imaging core loading and saving functions.

Description

Just put this unit in your uses clause instead of adding every unit that provides new file format support. Also new constants for SetOption/GetOption functions for new file formats are located here.

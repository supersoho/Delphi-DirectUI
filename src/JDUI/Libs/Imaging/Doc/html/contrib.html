Contributions

Several modifications and extensions contributed by Imaging users are included in release packages (located in (Imaging Root)\Extras\Contrib directory). Currently they are:

NameDescription
ExtraGIF Modification that provides extra functionality targeting GIF file format. Adds new self-animating TGraphic class and full frame information. Created by Sergey Galezdinov for QIP messenger. More info in included readme file.
NIF Loader File format extension that reads texture from version 3 NIF 3D models (ST: Bridge Commander). Author: Delfi.
hq2x for Pascal Pascal translation of great h2qx magnification filter. Demo showing how to use it with Imaging is included. Translated to Pascal by Jeremy Darling.

Also many thanks to all those who sent bug reports, fixes, diff patches, new functions, and feedback of any kind.

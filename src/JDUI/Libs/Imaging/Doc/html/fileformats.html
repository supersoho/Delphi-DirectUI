Supported File Formats

Image file format is device independent representation of image data. Images in these formats are usually stored as external files on hard disk. Many different file formats have been created but majority of the images stored anywhere are saved in one of a few most widely used formats. Main goal of Imaging's file format support is to fully support these widely used formats. This means successful loading and saving of all images stored in these formats. So if you have some image in one of supported formats and it can not be loaded by Imaging, please send it to me so I will be able to fix it.

Core File Formats
BMP Windows Bitmap File Format
JPEG File Format
PNG Portable Network Graphics File Format
GIF Graphics Interchange Format
TGA Targa File Format
DDS DirectDraw Surface File Format
MNG Multiple Network Graphics File Format
JNG JPEG Network Graphics File Format
Portable Maps: PBM, PGM, PPM, PAM, and PFM File Formats
Extras File Formats
